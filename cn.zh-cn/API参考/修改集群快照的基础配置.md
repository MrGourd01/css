# 修改集群快照的基础配置<a name="css_03_0030"></a>

## 功能介绍<a name="section874853215915"></a>

## URI<a name="section8763193210910"></a>

```
POST /v1.0/{project_id}/clusters/{cluster_id}/index_snapshot/setting
```
